# 🏝️ Treasure Island Adventure Game

A fun and interactive browser-based adventure game inspired by a Python text game. Built using **HTML, CSS, and JavaScript**, this project lets users make choices and explore different paths to find the hidden treasure.

## 🎮 Game Overview

You start your journey on a mysterious island. Every choice you make leads to different outcomes:

- Choose left or right at the crossroad
- Decide to wait or swim at the lake
- Pick the correct door to find the treasure

Each decision leads to either victory or game over!

## Features

- 🎨 Beautiful animated tropical background
- 🧭 Interactive decision-based gameplay
- 💎 Multiple endings (win or lose scenarios)
- 🔄 Restart game option
- 📱 Responsive design for mobile and desktop
- 🎯 Simple and beginner-friendly logic


## Technologies Used

- HTML5
- CSS3 (Glassmorphism + Animations)
- JavaScript (Game logic & DOM manipulation)

## How to Run the Project

1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html`
4. Run using **Live Server** or double-click the file
